# IoT_SmartAquarium

Admin Frontend using Vuejs

## Cài đặt

```bash
npm install
```

### Biên dịch và tự reload dành cho development

```bash
npm run serve
```

### Biên dịch và tối ưu dành cho production

```bash
npm run build
```

### Lints

```bash
npm run lint
```

## Sử dụng

- Truy cập trên trình duyệt: <http://localhost:3000/>
- Tài khoản admin: admin - 123456
